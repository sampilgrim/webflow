Webflow geometric pattern generator

Shapes and colours in collection lists are combined randomly in the grid using collection list random sorting. CSS mix blend modes are used to apply the colours to the shapes. No JavaScript is used to create the pattern.

The pattern updates on site export, or every 12 hours when Webflow updates its cache [1].

To modify shapes and colours, adjust classes on the Styles page.
